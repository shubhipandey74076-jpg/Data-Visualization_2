# Data-Visualization_2

📌 Project Overview
This project analyzes Netflix dataset using Python.
It focuses on understanding the distribution of genres across different countries using data visualization techniques.

🎯 Objectives
Analyze Netflix content data
Compare genres across countries
Create visual insights using graphs

🛠️ Tools & Technologies
Python 🐍
Pandas
Matplotlib
Seaborn
Google Colab
📂 Dataset
The dataset used in this project is a Netflix dataset in CSV format.
It contains information such as:
Country
Genre
Title
Release year
